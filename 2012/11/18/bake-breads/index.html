美的面包机制作提子面包攻略

发表于 2012-11-18   |   | 热度 ℃

我用的面包机是美的EHS10AH-PR. 一次最多做面包量500g.
面包机配有两个量杯和一个双头量勺。以下的杯或大勺小勺以标配的为准。
原材料：鲜牛奶，鸡蛋，黄油或植物油，盐，糖（白砂糖，绵白糖，红糖都可以，我偏好用红糖，会让面包有自然的焦糖色），安琪牌高活性酵母，面包粉，提子干。
步骤：

  1. 鸡蛋1个搅拌均匀，加入鲜牛奶，两者共计180ml. 注:鲜牛奶可以换成水和奶粉，用量与牛奶相同。

  2. 将混合均匀的液体倒入面包机，加入黄油1大勺。固体黄油，可以先用面包机自带的烘烤功能融化。黄油也可以用植物油代替。

  3. 加入盐0.5小勺（可以不加），红糖2大勺。搅拌均匀。因为是红糖，样子不太好看哈。如果是白砂糖的话，颜色会淡很多。

  4. 加入面包粉2杯。（我稍微减少了面包粉的量，加了1大勺泡打粉，2大勺玉米粉来调剂口味。）加入酵母粉1.5小勺。稍微搅拌一下，让酵母略均匀分布。 5.将面包桶放入面包机内，选择程序1的主食面包，需要等待的时间是3小时5分钟。
    中途会有bibi的提示音，提醒加入辅料。我一般放2大把提子干，喜欢的话可以多加点。还加了一小杯椰丝，但成品基本吃不出椰丝味。

坚果面包同提子面包制作完全相同，辅料改成自己爱吃的坚果就成。
香蕉面包，橙子面包也基本如法炮制。待我探索出最佳的固体和液体原材料的比例，再放上来。

烤好的成品见图片。为了更清楚的看到内部结构，刚出炉的面包被我扒了皮，哈哈。
